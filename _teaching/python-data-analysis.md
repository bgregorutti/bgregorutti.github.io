Practical hands-on workshops for researchers and professionals with little or no prior coding experience. Part of the [Sorbonne.AI](https://sorbonne.ai) educational program and [SCAI Education platform](https://scai-education.sorbonne-universite.fr).

This course is part of *Parcours UEIR - Intelligence artificielle appliquée à la médecine*.

## Topics

- Python basics
- Data manipulation with pandas
- Visualisation with matplotlib / seaborn
- Introduction to ML with scikit-learn

## Audiences

Medecine students

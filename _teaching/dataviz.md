Introduction to data visualization, concepts and practice with Python. Part of the Executive Education program of SCAI and Sorbonne Abu Dhabi

## Concepts covered

* Learn to design and build effective data visualizations
* Understand core visualization principles and chart selection
* Gain hands-on experience with key Python libraries
* Create interactive visualizations and dashboards
* Communicate insights clearly using real-world datasets

## Content

### Part 1 – Foundations of Data Visualization
What makes a good (and bad) visualization? Data types → chart types Overview of Python libraries Hands-on: Pandas and Matplotlib basics

### Part 2 – Foundations of Data manipulation and Pandas library
Learn how to load, explore, clean, transform, and summarize tabular data using the Pandas library

### Part 3 – Deep Dive: Matplotlib and Seaborn
Learn how to use Python libraries to create efficient plots with **Matplotlib** (basics) and **Seaborn** for Statistical Plots

### Part 4 – Interactive Plots with Bokeh
Learn how to use Python libraries to create interactive plots and dashboards with **Bokeh** library.

### Final Deliverables
* Analysis of a dataset and visualization
* Select tools and components
* Define a story and insights
* Implement an interactive dashboard using Bokeh
* Code and HTML dashboard submissions

## Audiences

Professionals in finance, ~10 students
